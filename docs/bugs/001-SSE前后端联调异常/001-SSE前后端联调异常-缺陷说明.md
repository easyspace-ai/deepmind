# SSE前后端联调异常 - 缺陷说明

## 1. 缺陷概述

- 缺陷编号：001
- 缺陷名称：SSE前后端联调异常
- 发现日期：2026-03-25
- 影响范围：
  - `internal/api/langgraph_handler.go`
  - `pkg/sse/deerflow/writer.go`
  - `frontend/src/core/threads/hooks.ts`
  - `frontend/src/core/api/stream-mode.ts`

## 2. 现象描述

- `frontend/` 聊天页面与后端联调时，流式消息不稳定
- 部分场景下消息不连续、结束异常或连接提前断开
- 页面依赖的 `title`、`todos`、`artifacts` 状态更新不稳定
- 前端开启 `streamResumable`，但后端 `joinStream` 未实现，存在恢复能力错配

## 3. 业务影响

- 影响聊天页面流式体验
- 影响多 Agent 页面与普通聊天页面的稳定联调
- 增加定位问题的成本，阻碍 SSE 能力继续扩展
