## 🎯 核心概念（分钟速览）

### Aggregate Root（聚合根）

**定义**: 持久化的业务对象，拥有唯一 ID、生命周期、不变量

```go
// ❌ 错误示例
agent.name = "new name"  // 直接修改字段

// ✅ 正确示例
agent.UpdateName("new name")  // 通过方法
if err := agent.Validate() {  // 维护不变量
    return err
}
agent.RaiseDomainEvent(&event)  // 发布事件
```

### Value Object（值对象）

**定义**: 不可变的、由属性定义的对象（没有 ID）

```go
// 值对象三要素
type Identity struct {
    name string  // 私有（不可变）
    role string
}

// ✅ 只读方法
func (i *Identity) Name() string { return i.name }

// ❌ 无 SetName()、无 public 字段
```

### Domain Event（领域事件）

**定义**: 发生在聚合根上的重要业务事件

```go
// ✅ 发布事件
if err := agent.AddCapability(cap); err != nil {
    return err
}
eventBus.Publish(ctx, &CapabilityAddedEvent{
    AgentID: agent.ID().String(),
    // ...
})

// ☑️ 异步处理（解耦）
eventBus.Subscribe("capability.added", func(ctx context.Context, event interface{}) {
    // 更新缓存、发送通知等
})
```

### Repository（仓储）

**定义**: 持久化的数据访问对象

```go
type AgentRepository interface {
    FindByID(ctx context.Context, id AgentID) (*Agent, error)
    Save(ctx context.Context, agent *Agent) error
    FindByUserCode(ctx context.Context, userCode string) ([]*Agent, error)
}

// ✅ 多租户检查内置
// ✅ 返回聚合根，不是 DTO
// ✅ 处理并发（乐观锁）
```

## ⚙️ 常用代码模式

### Pattern 1: 创建聚合根

```go
// ✅ 使用工厂函数
agent := NewAgent(userCode, identity)  // 返回 *Agent

// ✅ 初始化事件已发布
events := agent.GetUncommittedEvents()  // len(events) == 1
```

### Pattern 2: 业务规则（最重要！）

```go
// ✅ 聚合根内部的业务逻辑
func (a *Agent) AddCapability(cap *Capability) error {
    // 第一步：验证
    if !a.canAddCapability(cap) {
        return fmt.Errorf("cannot add: %w", reason)
    }
    
    // 第二步：修改状态
    a.capabilities.Add(cap)
    a.version++
    
    // 第三步：发布事件
    a.RaiseDomainEvent(&CapabilityAddedEvent{
        AgentID:    a.id.String(),
        Capability: cap,
        AddedAt:    time.Now(),
    })
    
    return nil
}

// ❌ 错误做法：业务逻辑在 Service 层
// service.AddCapabilityToAgent(agent, cap)  // 不要这样做！
```

### Pattern 3: 查询方法

```go
// ✅ 只读，无副作用
func (a *Agent) HasCapability(name string) bool {
    return a.capabilities.Contains(name)
}

func (a *Agent) GetCapabilities() []*Capability {
    return a.capabilities.All()
}

// ❌ 不要返回内部集合的指针
// return a.capabilities.Items  // 违反封装
```

### Pattern 4: 值对象相等性

```go
// ✅ 值对象有 Equals 方法
func (i *Identity) Equals(other *Identity) bool {
    return i.name == other.name && i.role == other.role
}

// 用法
if current.Equals(newIdentity) {
    return nil  // 无需更新
}
```

### Pattern 5: 多租户检查

```go
// ✅ 每个聚合根都有 UserCode
type Agent struct {
    id       AgentID
    userCode string  // 关键！不能忘记
    // ...
}

// ✅ 仓储强制多租户隔离
func (r *AgentRepository) FindByID(ctx context.Context, id AgentID) (*Agent, error) {
    userCode := ctx.Value("user_code").(string)
    // SELECT * FROM agents WHERE id = ? AND user_code = ?
    return r.db.Where("id = ? AND user_code = ?", id, userCode).First()
}
```

## 🔒 多租户隔离检查清单

**在写任何 CRUD 代码时检查**：

- [ ] 聚合根是否有 `userCode` 字段？
- [ ] 仓储的查询是否都过滤 `userCode`？
- [ ] 是否从 context 获取 `userCode`？
- [ ] 是否验证了用户权限？

```go
// ❌ 危险代码
agent, _ := repo.FindByID(ctx, agentID)  // 无租户检查！

// ✅ 安全代码
agent, _ := repo.FindByID(ctx, agentID, fromContext(ctx).UserCode)
```

## 🧪 测试模板

### 聚合根测试

```go
func TestAgentNewAgent(t *testing.T) {
    identity := NewIdentity("AI", "desc", "assistant")
    agent := NewAgent("user-1", identity)
    
    // 1. 检查创建
    assert.NotNil(t, agent)
    assert.NotEmpty(t, agent.ID())
    
    // 2. 检查事件
    events := agent.GetUncommittedEvents()
    assert.Len(t, events, 1)
    assert.IsType(t, (*AgentCreatedEvent)(nil), events[0])
}

func TestAgentAddCapabilityError(t *testing.T) {
    agent := NewAgent("user-1", identity)
    cap := &Capability{Name: "search"}
    
    err := agent.AddCapability(cap)
    assert.NoError(t, err)
    
    // 重复添加应该失败
    err = agent.AddCapability(cap)
    assert.Error(t, err)
}
```

### 仓储测试

```go
func TestAgentRepositoryFindByID(t *testing.T) {
    repo := setupRepository(t)
    agent := NewAgent("user-1", identity)
    
    // 保存
    _ = repo.Save(context.Background(), agent)
    
    // 查询
    found, err := repo.FindByID(context.Background(), agent.ID())
    assert.NoError(t, err)
    assert.Equal(t, agent.ID(), found.ID())
    assert.Equal(t, "user-1", found.UserCode())
}
```

## ❌ 常见错误

| 错误 | 后果 | 修复 |
|------|------|------|
| 直接修改聚合根字段 | 违反封装、难以追踪 | 使用方法 + 事件 |
| 业务逻辑在 Service | 聚合根变贫血 | 移动到聚合根 |
| 忽略 UserCode | 多租户隔离失效 | 每个查询都过滤 |
| 事件发布到数据库 | 无法查询 | 使用事件总线 |
| 值对象有 Setter | 不可变性破坏 | 删除 Setter |
| 聚合根包含太多 | 测试复杂 | 分解为多个聚合 |
