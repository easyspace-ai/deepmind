### 1. 030-DDD重构方案-完整计划.md

**类型**: 新增  
**规模**: ~2500 行  
**覆盖范围**: 完整的 DDD 重构方案

#### 核心内容

```
Part 1: 7 个 Bounded Contexts 定义
├─ Agent Orchestration Context
├─ Session Management Context
├─ Message & Conversation Context
├─ Channel Communication Context
├─ User & Auth Context
├─ LLM Provider Context
└─ MCP Server Context (重新设计)

Part 2: 领域事件与中间件链
├─ 领域事件定义
├─ 事件总线设计
└─ 中间件链（参考 DeerFlow）

Part 3: Agent 系统重构
├─ Agent 执行引擎
├─ 提示词系统（DeerFlow 风格）
└─ 沙箱系统（虚拟路径）

Part 4: 消息处理流程重构
├─ 完整消息流程图
└─ 消息处理服务

Part 5: 仓储与持久化
├─ Repository 接口
└─ GORM 实现

Part 6: 依赖注入与启动
├─ DI 容器设计
└─ 启动流程

Part 7: 渐进式迁移策略
├─ 兼容层
└─ 分阶段迁移计划
```

#### 关键贡献

- ✅ 定义了所有 7 个 Bounded Context 的聚合根
- ✅ 提供了完整的代码示例（Go 语言）
- ✅ 设计了 DeerFlow 集成方案
- ✅ 规划了 6 周执行计划
