## 1. 需求背景

当前对话记录页面仅使用表格形式展示数据，虽然包含了 trace_id、span_id、parent_span_id、token 消耗等丰富信息，但这些数据的潜力未被充分利用。用户难以直观理解对话的调用链路、会话上下文和统计趋势。

## 2. 需求目标

增强对话记录页面的数据可视化能力，提供多维度数据展示和分析功能：
1. **对话链路可视化** - 基于 trace/span 展示调用树
2. **会话对话视图** - 聊天形式展示同一会话的消息流
3. **统计分析仪表盘** - Token 消耗、Agent 使用等统计图表
4. **增强筛选功能** - 多维度数据筛选

## 3. 功能需求

### 3.1 对话链路可视化

**目标**: 利用 trace_id + span_id + parent_span_id 构建调用树，直观展示一次完整对话的执行流程。

**功能点**:
- 在操作列增加"查看链路"按钮
- 弹窗展示树形/流程图结构的调用链路
- 每个节点显示：角色、事件类型、token消耗、耗时
- 支持节点点击查看详情
- 高亮显示关键路径

**数据需求**:
- 后端需提供按 trace_id 查询接口（已有）
- 需要计算节点间的时间差作为耗时

### 3.2 会话对话视图

**目标**: 按 session_key 聚合消息，以聊天界面形式展示对话上下文。

**功能点**:
- 在操作列增加"查看会话"按钮
- 新页面/弹窗展示聊天式对话记录
- 左右布局：用户消息右对齐，助手消息左对齐
- 显示每条消息的 token 消耗
- 支持展开查看完整内容
- 显示会话统计：消息数、总token、对话时长

**数据需求**:
- 后端需支持按 session_key 分页查询（已有）

### 3.3 统计分析仪表盘

**目标**: 提供对话数据的统计分析能力。

**功能点**:
- 新增"统计分析"页面入口
- 展示以下图表：
  - Token 消耗趋势（折线图，按日/周/月）
  - Agent 使用分布（柱状图/饼图）
  - Channel 来源分布（饼图）
  - 角色消息分布（饼图）
  - 平均对话轮次统计
  - 响应时间分布（直方图）
- 支持时间范围筛选
- 支持 Agent/Channel 筛选

**数据需求**:
- 后端需提供聚合统计接口

### 3.4 增强筛选功能

**目标**: 提升对话记录列表的筛选能力。

**功能点**:
- 时间范围选择器（开始时间-结束时间）
- Agent 多选下拉框
- Channel 多选下拉框
- 角色类型多选（user/assistant/system/tool/tool_result）
- Token 数量范围输入
- 重置筛选按钮

**UI 要求**:
- 筛选栏可折叠
- 已选筛选条件以标签形式展示
- 支持快速清除单个筛选条件

### 3.5 链路顺序一致性修正

**目标**: 使“查看链路”中的事件展示顺序与“查看对话”保持一致，并修正工具调用相关节点的层级关系。

**功能点**:
- 链路视图中的基础时间顺序与“查看对话”一致
- 同时间戳下，`tool` 节点必须显示在 `tool_result` 之前
- 同时间戳下，`llm_call_end` 必须显示在 `tool_completed` 之前
- `tool_result` 节点应展示为对应 `tool` 节点的子节点
- 不允许因父子关系构建错误导致事件丢失或仅显示单节点

**验收标准**:
- [ ] 同一 trace 在“查看链路”与“查看对话”的时间顺序一致
- [ ] 工具节点在上，工具结果节点在下，且层级关联正确
- [ ] LLM 结束事件可见，且显示顺序早于 tool_completed
- [ ] 链路视图不会出现只剩用户消息的异常场景

## 4. UI 需求

### 4.1 页面布局

```
对话记录页面
├── 页面标题 + 统计概览卡片
├── 筛选栏（可折叠）
│   ├── 时间范围 | Agent多选 | Channel多选 | 角色多选 | Token范围
│   └── 已选条件标签 + 重置按钮
├── 数据展示区（标签页切换）
│   ├── 列表视图（现有表格）
│   ├── 会话视图（按session分组）
│   └── 链路视图（按trace分组）
└── 操作按钮
    ├── 导出数据
    └── 查看统计仪表盘
```

### 4.2 弹窗设计

**链路详情弹窗**:
```
+------------------------------------------+
| 对话链路 - trace_abc123          [X]     |
+------------------------------------------+
| 总耗时: 2.3s | 总Token: 1,234            |
+------------------------------------------+
|                                          |
|  [User]          [Assistant]             |
|    |                  |                  |
|    |----request----->|                   |
|    |                  |----tool_call--->|
|    |                  |<---tool_result---|
|    |<----response----|                   |
|                                          |
+------------------------------------------+
```

**会话对话弹窗**:
```
+------------------------------------------+
| 会话详情 - session_xyz456        [X]     |
+------------------------------------------+
| 消息: 8 | 总Token: 3,456 | 时长: 5min    |
+------------------------------------------+
|                                          |
|         你好，帮我查下天气               |
|                   [user] 12 Tokens       |
|  -------------------------------------   |
|  好的，我来为您查询北京天气...             |
|  [assistant] 156 Tokens                  |
|                                          |
+------------------------------------------+
```

## 5. 验收标准

- [ ] 对话链路可视化可展示完整调用树
- [ ] 会话对话视图以聊天形式展示消息流
- [ ] 统计分析页面展示6种图表
- [ ] 增强筛选支持所有指定维度
- [ ] 筛选条件可正确重置
- [ ] 响应式设计，适配不同屏幕
- [ ] 前端交互测试通过
