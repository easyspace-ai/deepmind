## 1. 实现概述

在 LLM Provider 管理页面增加了嵌入式模型（Embedding Model）配置管理功能，支持查看、添加、编辑、删除嵌入模型，以及设置默认嵌入模型。

## 2. 实现内容

### 2.1 后端 API 使用

复用已有后端接口：
- `GET /api/v1/providers/:id/embedding` - 获取嵌入模型配置
- `PUT /api/v1/providers/:id/embedding` - 更新嵌入模型配置

### 2.2 前端 API 扩展

**文件**: `web/src/api/providers.ts`

新增方法：
```typescript
getEmbeddingModels: (id: number) => Promise<{ embedding_models, default_embedding_model, has_embedding_models }>
updateEmbeddingModels: (id: number, data) => Promise<void>
```

### 2.3 类型定义扩展

**文件**: `web/src/types/index.ts`

新增：
- `EmbeddingModelInfo` 接口
- `LLMProvider` 增加 `embedding_models` 和 `default_embedding_model` 字段

### 2.4 Provider 页面增强

**文件**: `web/src/pages/Providers.tsx`

新增功能：
1. **表格列**: 新增"嵌入模型"列，显示配置状态（已配置模型数量/未配置）
2. **操作按钮**: 新增"嵌入模型"配置按钮
3. **配置弹窗**: 独立的嵌入模型管理弹窗，包含：
   - 嵌入模型列表表格（ID、名称、维度、默认标识）
   - 设为默认/删除操作
   - 添加新模型表单（ID、名称、维度）
   - 保存/取消按钮

### 2.5 状态管理

新增状态：
```typescript
const [embeddingModalVisible, setEmbeddingModalVisible] = useState(false);
const [embeddingModels, setEmbeddingModels] = useState<EmbeddingModelInfo[]>([]);
const [defaultEmbeddingModel, setDefaultEmbeddingModel] = useState('');
const [embeddingProvider, setEmbeddingProvider] = useState<LLMProvider | null>(null);
const [embeddingInput, setEmbeddingInput] = useState({ id: '', name: '', dimensions: 1536 });
```

### 2.6 核心方法

- `openEmbeddingModal` - 打开配置弹窗并加载数据
- `handleAddEmbeddingModel` - 添加新模型（含重复校验）
- `handleRemoveEmbeddingModel` - 删除模型（自动清除默认设置）
- `handleSetDefaultEmbeddingModel` - 设置默认模型
- `handleSaveEmbeddingModels` - 保存配置到后端

## 4. 关键实现点

### 4.1 UI 布局

```
配置嵌入模型 - {provider_name}
├─ 嵌入模型列表表格
│  ├─ 模型ID | 名称 | 维度 | 默认 | 操作
│  └─ 每行：设为默认按钮 | 删除按钮
│
└─ 添加新模型区域（灰色背景）
   ├─ 模型ID输入框 | 名称输入框 | 维度输入框 | 添加按钮
   └─ 维度默认值：1536
```

### 4.2 交互逻辑

1. 点击"嵌入模型"按钮 → 加载当前配置 → 打开弹窗
2. 添加模型 → 本地验证（ID不能重复）→ 添加到列表
3. 删除模型 → 如果是默认模型则清除默认设置
4. 保存 → 调用 API → 刷新 Provider 列表

### 4.3 数据格式

```typescript
interface EmbeddingModelInfo {
  id: string;        // 如：text-embedding-3-small
  name: string;      // 如：Text Embedding 3 Small
  dimensions: number; // 如：1536
}
```

## 6. 已知限制与待改进点

### 6.1 当前限制

1. 嵌入模型列表不分页（通常模型数量较少）
2. 维度输入框使用数字输入，但无常用维度预设

### 6.2 待改进点

1. 可以添加常用嵌入模型预设（如 OpenAI 的 text-embedding-3-small/large）
2. 支持从 API 自动获取嵌入模型列表
3. 支持嵌入模型测试功能

## 8. 使用说明

1. 进入 LLM Provider 管理页面
2. 点击 Provider 行中的"嵌入模型"按钮
3. 在弹窗中：
   - 查看已有嵌入模型列表
   - 点击"设为默认"设置默认模型
   - 点击"删除"移除模型
   - 在底部表单填写新模型信息并点击"添加"
4. 点击"保存"提交配置
