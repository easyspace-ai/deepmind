**代码示例**:
```typescript

  localStorage.removeItem('user');

  window.location.href = '/login';
}
```

### 4. Token 存储安全优化 ⏸️ 待实施

**当前状态**:
- Token 仍存储在 localStorage
- 存在 XSS 风险

**方案评估**:

| 方案 | 可行性 | 工作量 | 优先级 |
|------|--------|--------|--------|
| httpOnly Cookie | 需后端支持 | 中等 | P2 |
| Token 刷新机制 | 前后端均需改动 | 较大 | P2 |
| 保持现状 | - | - | - |

**建议**:
- 短期：保持 localStorage，确保前端 XSS 防护
- 长期：迁移到 httpOnly Cookie

---

### 5. API 层代码生成调研 ⏸️ 待实施

**调研结果**:

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| 手写维护 | 灵活，当前使用 | 容易不同步 | ⭐⭐⭐ |
| OpenAPI Generator | 自动生成 | 需要维护规范 | ⭐⭐⭐⭐ |
| orval + MSW | 生成+Mock | 工具链复杂 | ⭐⭐ |

**建议**:
- 当前阶段继续使用手写维护
- 待 API 稳定后引入 OpenAPI Generator
