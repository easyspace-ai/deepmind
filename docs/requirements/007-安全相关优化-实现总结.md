## 实现详情

### 1. JWT 密钥硬编码修复 ✅

**实现文件**: `internal/api/auth.go`

**主要改动**:
- 新增 `getJWTSecret()` 函数从环境变量 `JWT_SECRET` 读取密钥
- 新增 `getTokenDuration()` 函数从环境变量 `JWT_TOKEN_DURATION` 读取有效期，默认 7 天
- 新增 `ValidateJWTConfig()` 函数用于启动时验证 JWT 配置
- 修改 `ParseToken` 函数使用动态获取的密钥

**代码示例**:
```go
func getJWTSecret() []byte {
    secret := os.Getenv("JWT_SECRET")
    if secret == "" {
        return []byte("nanobot-dev-secret-key-change-in-production")
    }
    return []byte(secret)
}

func ValidateJWTConfig() error {
    secret := os.Getenv("JWT_SECRET")
    if secret == "" {
        return fmt.Errorf("JWT_SECRET environment variable is required")
    }
    if len(secret) < 32 {
        return fmt.Errorf("JWT_SECRET must be at least 32 characters long")
    }
    return nil
}
```

**配置模板** (`.env.example`):
```bash
JWT_SECRET=your-super-secret-jwt-key-min-32-chars
JWT_TOKEN_DURATION=168h
```

**状态**: 已实现，生产环境需配置 `JWT_SECRET` 环境变量

### 2. LLM Adapter API 密钥硬编码修复 ✅

**实现文件**: `memory/service/llm_adapter.go`, `internal/app/memory.go`

**主要改动**:
- `systemLLMClient` 结构体新增 `db *gorm.DB` 字段
- 新增 `getAPIConfigFromDB()` 方法从数据库 Provider 表读取配置
- 修改 `NewSystemLLMClient()` 函数签名，传入数据库连接
- 支持默认 Provider 选择和模型匹配

**代码示例**:
```go
func (c *systemLLMClient) getAPIConfigFromDB(modelName string) (string, string, error) {
    var provider models.LLMProvider
    err := c.db.Where("is_default = ? AND is_active = ?", true, true).
        First(&provider).Error
    if err != nil {
        // 回退到任意活跃 Provider
        err = c.db.Where("is_active = ?", true).
            Order("priority DESC").
            First(&provider).Error
    }
    return provider.APIKey, provider.APIBase, nil
}
```

**状态**: 已实现，API Key 从数据库 Provider 表读取
