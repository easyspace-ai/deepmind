# Skills 管理功能需求

## 背景与目标

当前 Skills（技能）仅在 Agent 管理界面下通过下拉列表选择绑定关系，缺少专门的 Skills 管理界面来查看和管理系统中所有可用的技能。

## 功能需求

### 核心功能

1. **技能列表展示**
   - 显示所有可用技能
   - 区分技能来源（workspace 工作区 / builtin 内置）
   - 显示技能名称和描述

2. **技能详情查看**
   - 查看 SKILL.md 完整内容
   - 查看绑定的 Agent 列表

3. **界面要求**
   - 与现有管理界面风格一致
   - 支持响应式布局

## 技能来源

- **workspace**: 项目 `skills/` 目录下的技能
- **builtin**: 程序内置的技能

## 用户场景

1. 管理员想查看系统有哪些可用技能
2. 管理员想了解某个技能的具体功能
3. 管理员想查看某个技能被哪些 Agent 使用

## 验收标准

- [ ] 页面能正确显示所有 Skills 列表
- [ ] 每个 Skill 显示名称和描述
- [ ] 点击 Skill 可查看详情（SKILL.md 内容）
- [ ] 能查看绑定该技能的 Agent 列表
- [ ] 界面风格与其他管理页面一致
