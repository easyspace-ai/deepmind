### 1. 分页逻辑提取为通用函数 ✅

**实现文件**: `internal/utils/pagination/pagination.go` (新增)

**主要改动**:

- 实现 `NormalizeDefault()` 使用默认值（20/1000）
- 实现 `CalculateOffset()` 和 `CalculateTotalPages()` 辅助函数
- 实现 `NewResult()` 创建完整分页结果对象

**代码示例**:
```go
package pagination

// Normalize 规范化分页参数
func Normalize(page, pageSize, defaultSize, maxSize int) (int, int) {
    if page < 1 {
        page = 1
    }
    if pageSize < 1 {
        pageSize = defaultSize
    }
    if pageSize > maxSize {
        pageSize = maxSize
    }
    return page, pageSize
}

// CalculateOffset 计算数据库偏移量
func CalculateOffset(page, pageSize int) int {
    return (page - 1) * pageSize
}
```

**应用范围**:
- `internal/service/conversation/service.go`: `ListBySessionKey`, `ListByTimeRange`, `ListRecent`

**优化效果**:
- 减少重复代码约 30 行
- 统一分页参数校验逻辑
- 支持自定义默认值和最大值

### 2. 数据迁移脚本 Provider 映射修复 ✅

**实现文件**: `scripts/migrate_data.go`

**主要改动**:
- 新增 `OldProvider` 结构体读取旧数据的 UserID 字段
- 修改 `migrateProviders()` 函数，先查询 Provider 与 User 的关联关系
- 使用正确的 UserID 映射 UserCode，而非假设 Provider ID 等于 User ID

**代码示例**:
```go
type OldProvider struct {
    ID     uint
    UserID uint // 关联的用户ID
}

func migrateProviders(oldDB, newDB *gorm.DB) error {
    // 获取旧Provider的UserID映射
    var oldProviders []OldProvider
    oldDB.Table("llm_providers").Select("id, user_id").Find(&oldProviders)
    providerUserMap := make(map[uint]uint)
    for _, op := range oldProviders {
        providerUserMap[op.ID] = op.UserID
    }

    for _, p := range providers {
        // 使用Provider关联的UserID查找UserCode
        if userID, ok := providerUserMap[p.ID]; ok {
            p.UserCode = userCodeMap[userID]
        }
    }
}
```

**优化效果**:
- 修复数据迁移时的错误映射问题
- 确保 Provider 与正确的 User 关联

### 3. Repository 层批量查询优化 ✅

**实现文件**: `internal/repository/agent.go`, `internal/repository/channel.go`

**主要改动**:
- `AgentRepository` 接口新增 `GetByAgentCodes(codes []string) ([]*Agent, error)` 方法
- `ChannelRepository` 接口新增 `GetByChannelCodes(codes []string) ([]*Channel, error)` 方法
- 使用 SQL `IN` 查询替代循环单次查询

**代码示例**:
```go
// AgentRepository 接口
GetByAgentCodes(codes []string) ([]*models.Agent, error) // 批量查询

// 实现
func (r *agentRepository) GetByAgentCodes(codes []string) ([]*models.Agent, error) {
    if len(codes) == 0 {
        return []*models.Agent{}, nil
    }
    var agents []*models.Agent
    if err := r.db.Where("agent_code IN ?", codes).Find(&agents).Error; err != nil {
        return nil, fmt.Errorf("批量获取 Agent 失败: %w", err)
    }
    return agents, nil
}
```

**预期效果**:
- N+1 查询优化为 1 次批量查询
- 大幅提升对话记录列表页性能
- 保持现有 API 兼容性
