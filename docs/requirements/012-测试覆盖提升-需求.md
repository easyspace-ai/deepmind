## 测试规范

### 单元测试规范
```go
// 命名: Test<函数名>_<场景>_<预期>
func TestCodeService_GenerateUserCode_Success(t *testing.T) {
    // Arrange
    service := NewCodeService()

    // Act
    code, err := service.GenerateUserCode()

    // Assert
    require.NoError(t, err)
    assert.True(t, strings.HasPrefix(code, "usr_"))
    assert.Len(t, code, 14) // 4 prefix + 10 random
}
```

### Mock 规范
- 使用 `github.com/stretchr/testify/mock`
- 或 `go.uber.org/mock`

### 集成测试规范
```go
// +build integration

func TestChannelRepository_Create(t *testing.T) {
    // 使用内存数据库
    db, err := gorm.Open(sqlite.Open(":memory:"), &gorm.Config{})

    // 自动迁移

    repo := NewChannelRepository(db)
    // ... 测试代码

```
