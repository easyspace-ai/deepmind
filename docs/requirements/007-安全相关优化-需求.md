# 需求 007: 安全相关优化

## 背景

在代码审查中发现多处安全相关的硬编码和未完成的功能，需要进行统一修复。

## 需求列表

### 1. JWT 密钥硬编码修复

**现状**
- 文件: `internal/api/auth.go:12`
- JWT 密钥硬编码在代码中: `"nanobot-secret-key-change-in-production"`

**需求**
- JWT 密钥应从环境变量 `JWT_SECRET` 读取
- 如果环境变量未设置，启动时应该报错而不是使用默认值
- Token 有效期可配置（环境变量 `JWT_TOKEN_DURATION`，默认 7天）

**验收标准**
- [ ] JWT 密钥从环境变量读取
- [ ] 未设置环境变量时服务启动失败并给出明确错误
- [ ] 添加相应的配置文档
