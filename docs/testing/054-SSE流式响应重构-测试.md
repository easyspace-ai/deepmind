- 验证 SSE 后端改造后相关 Go 包可以正常编译
- 验证 DeerFlow Go client 能兼容标准事件名
- 验证前端改动未引入额外的恢复能力错配

## 2. 测试范围

- `pkg/sse`
- `pkg/sse/deerflow`
- `pkg/deerflow`
- `internal/api`
- `frontend/src/core/threads/hooks.ts`

## 4. 风险说明

- 尚未覆盖真实浏览器端到端联调
- 尚未覆盖 `joinStream` 恢复路径，因为后端仍未实现该能力
